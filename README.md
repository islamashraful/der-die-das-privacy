# Der Die Das Privacy Policy

Static privacy policy site for **Der Die Das: Deutsch Lernen**.

## GitHub Pages

1. Create a GitHub repository named `der-die-das-privacy`.
2. Push this local repository to GitHub.
3. In the GitHub repository, open **Settings > Pages**.
4. Set the source to **Deploy from a branch**.
5. Select the `main` branch and `/root`.
6. Save the Pages settings.

The published site will usually be available at:

```text
https://<github-username>.github.io/der-die-das-privacy/
```

The `/privacy/` path redirects to the root policy page.
